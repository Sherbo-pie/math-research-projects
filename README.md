# Math Research Projects — Shirsha Pattanaik

Academic research projects completed at Mahindra University 
as part of the B.Tech. Computational Mathematics program.

## 1. How Group Theory Secures Online Banking & E-Commerce
**Course:** Algebra | **Date:** November 2025

Formal study of RSA cryptography through group theory. Covers 
mathematical proof of RSA correctness using Euler's theorem and 
Lagrange's theorem, security analysis via the integer factorisation 
problem, and a Python implementation of RSA from scratch.

## 2. Music Genre Classification Using Frequency Domain Analysis
**Course:** Signals & Systems | **Date:** November 2025

ML-based genre classifier built on the GTZAN dataset (1000 clips, 
10 genres). Extracted 26 MFCC features using librosa; benchmarked 
KNN, SVM, ANN, and Gradient Boosting — ANN achieved best accuracy 
at 68.5%.
